## Description

This is a demonstration instance of the Nest.js Framework that can be seamlessly integrated into any existing Express.js-based RESTful API codebases

## Installation

```bash
$ ./install.sh #github repository url with Express.js API#
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
